พัฒนาระบบจัดการข้อมูล ด้วยภาษา Golang
=========================================
- มีหน้า login                                               
- แสดงข้อมูลได้                                           
- เพิ่มข้อมูล (มีป้องกัน sql injection)           
- แก้ไขข้อมูล (มีป้องกัน sql injection)       
- ลบข้อมูล          
##tech stack##
- bootstrap
- sweet alert
- database ใช้ Mysql                                         
                          
ฟังก์ชันหลักของระบบ
🔐 เข้าสู่ระบบ (Login) แยกสิทธิ์ผู้ใช้งาน (Admin / Staff)

📦 จัดการสินค้า (Product Management)	เพิ่ม / ลบ / แก้ไขข้อมูลสินค้า เช่น รหัส, ชื่อ, ราคา, หมวดหมู่, หน่วยนับ

➕ รับสินค้าเข้า (Stock In) บันทึกจำนวนที่นำเข้า, วันที่, แหล่งที่มา

➖ เบิกสินค้าออก (Stock Out)	บันทึกการเบิกใช้หรือจำหน่าย

🔎 ค้นหาสินค้า	ค้นหาด้วยชื่อ / รหัส / หมวดหมู่

📉 แจ้งเตือนสินค้าคงเหลือต่ำ	แสดงรายการสินค้าที่ต่ำกว่าค่าขั้นต่

📅 รายงานและสถิติ (Reports)	แสดงยอดคงเหลือ, สินค้าขายดี, รายการเข้า–ออกตามช่วงเวลา

👥 จัดการผู้ใช้งาน (User Management)	เพิ่ม / ลบ / แก้ไขสิทธิ์ผู้ใช้ระบบ
