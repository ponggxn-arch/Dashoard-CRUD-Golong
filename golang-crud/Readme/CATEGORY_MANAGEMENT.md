# 📁 Category Management Feature - คู่มือการจัดการหมวดหมู่สินค้า

## ✨ ฟีเจอร์ใหม่: จัดการหมวดหมู่สินค้า

ระบบได้เพิ่มความสามารถในการจัดการหมวดหมู่สินค้าอย่างสมบูรณ์แล้ว!

### 🎯 ฟังก์ชันที่เพิ่มเข้ามา

| ฟังก์ชัน | คำอธิบาย |
|---------|----------|
| ✅ **ดูรายการหมวดหมู่** | แสดงหมวดหมู่สินค้าทั้งหมดในระบบ |
| ✅ **เพิ่มหมวดหมู่ใหม่** | สร้างหมวดหมู่สินค้าใหม่พร้อมรายละเอียด |
| ✅ **แก้ไขหมวดหมู่** | แก้ไขชื่อและรายละเอียดหมวดหมู่ |
| ✅ **ลบหมวดหมู่** | ลบหมวดหมู่ (ถ้าไม่มีสินค้าในหมวดหมู่) |
| ✅ **ตรวจสอบการใช้งาน** | ป้องกันการลบหมวดหมู่ที่มีสินค้าอยู่ |

---

## 📍 การเข้าถึงฟีเจอร์

### วิธีที่ 1: ผ่านเมนูหลัก
1. คลิกเมนู **"สินค้า"** ในแถบนำทาง
2. เลือก **"จัดการหมวดหมู่"**
3. หรือไปที่ URL: `http://localhost:8080/categories`

### วิธีที่ 2: ลิงก์โดยตรง
```
http://localhost:8080/categories
```

---

## 🔧 การใช้งาน

### 1️⃣ เพิ่มหมวดหมู่ใหม่

**ขั้นตอน:**
1. คลิกปุ่ม **"เพิ่มหมวดหมู่ใหม่"** สีขาว
2. กรอกข้อมูล:
   - **ชื่อหมวดหมู่*** (บังคับ) - เช่น: อุปกรณ์คอมพิวเตอร์, วัสดุก่อสร้าง
   - **รายละเอียด** (ไม่บังคับ) - อธิบายเกี่ยวกับหมวดหมู่
3. คลิก **"บันทึก"**
4. ระบบจะแจ้งเตือนความสำเร็จและกลับไปหน้ารายการ

**ตัวอย่างหมวดหมู่:**
```
- อิเล็กทรอนิกส์ (สินค้าอิเล็กทรอนิกส์และอุปกรณ์ไฟฟ้า)
- เครื่องใช้สำนักงาน (อุปกรณ์และเครื่องใช้สำนักงาน)
- อะไหล่ (อะไหล่และชิ้นส่วนต่างๆ)
- เครื่องมือ (เครื่องมือช่างและอุปกรณ์)
- วัสดุสิ้นเปลือง (วัสดุที่ใช้แล้วหมดไป)
```

---

### 2️⃣ แก้ไขหมวดหมู่

**ขั้นตอน:**
1. คลิกปุ่ม **แก้ไข** (ไอคอนดินสอสีเหลือง) ในรายการที่ต้องการ
2. แก้ไขข้อมูล:
   - ชื่อหมวดหมู่
   - รายละเอียด
3. คลิก **"บันทึก"**

**หมายเหตุ:**
- การแก้ไขหมวดหมู่จะไม่กระทบกับสินค้าที่อยู่ในหมวดหมู่นั้น
- สินค้าทั้งหมดจะยังคงอยู่ในหมวดหมู่เดิม

---

### 3️⃣ ลบหมวดหมู่

**ขั้นตอน:**
1. คลิกปุ่ม **ลบ** (ไอคอนถังขยะสีแดง)
2. ยืนยันการลบในหน้าต่างแจ้งเตือน
3. ระบบจะตรวจสอบ:
   - ✅ **ถ้าไม่มีสินค้าในหมวดหมู่** → ลบได้
   - ❌ **ถ้ามีสินค้าในหมวดหมู่** → แจ้งเตือนไม่สามารถลบได้

**ข้อควรระวัง:**
- หมวดหมู่ที่มีสินค้าอยู่จะไม่สามารถลบได้
- ต้องย้ายหรือลบสินค้าในหมวดหมู่นั้นก่อน
- หมวดหมู่ที่ลบแล้วจะไม่สามารถกู้คืนได้

## 📋 รายละเอียดหน้าจอ

### หน้ารายการหมวดหมู่
แสดงตาราง 5 คอลัมน์:
1. **ID** - รหัสหมวดหมู่ (สีน้ำเงิน)
2. **ชื่อหมวดหมู่** - ตัวหนา
3. **รายละเอียด** - คำอธิบาย
4. **วันที่สร้าง** - รูปแบบ DD/MM/YYYY
5. **จัดการ** - ปุ่มแก้ไข (เหลือง) และลบ (แดง)

### หน้าฟอร์ม (เพิ่ม/แก้ไข)
- **ชื่อหมวดหมู่** - ช่องกรอกข้อความ (บังคับ)
- **รายละเอียด** - Textarea 4 บรรทัด (ไม่บังคับ)
- **ข้อความแนะนำ** - กล่องสีน้ำเงินอ่อนอธิบายการใช้งาน
- ปุ่ม: **ยกเลิก** (กลับหน้ารายการ) และ **บันทึก**

## 🔧 API Endpoints

## 💡 Tips & Best Practices

### ✅ ควรทำ
1. **ตั้งชื่อหมวดหมู่ที่ชัดเจน** - ง่ายต่อการค้นหา
2. **ใส่รายละเอียด** - ช่วยให้เข้าใจวัตถุประสงค์
3. **จัดกลุ่มอย่างสมเหตุสมผล** - ไม่ควรมีหมวดหมู่มากเกินไป
4. **ตรวจสอบก่อนลบ** - แน่ใจว่าไม่มีสินค้าในหมวดหมู่

### ❌ ไม่ควรทำ
1. **ตั้งชื่อซ้ำกัน** - ระบบจะไม่อนุญาต
2. **ลบหมวดหมู่ที่มีสินค้า** - ระบบจะป้องกัน
3. **ใช้ชื่อยาวเกินไป** - ควรไม่เกิน 50 ตัวอักษร
