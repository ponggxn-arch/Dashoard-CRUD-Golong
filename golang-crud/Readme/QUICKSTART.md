# 🚀 Quick Start Guide - ระบบจัดการสินค้าคงคลัง

## ⚡ เริ่มต้นใช้งานด่วน (5 นาที)

### ขั้นตอนที่ 1: เตรียมฐานข้อมูล
```bash
# เข้า MySQL
mysql -u root -p

# สร้างและใช้งานฐานข้อมูล
CREATE DATABASE golang_crud;
USE golang_crud;

# นำเข้าโครงสร้างตาราง
SOURCE inventory_database.sql;
```

### ขั้นตอนที่ 2: รันโปรแกรม
```bash
# วิธีที่ 1: ใช้ PowerShell Script
.\start.ps1

# วิธีที่ 2: รันด้วย Go
go run main.go
```

### ขั้นตอนที่ 3: เข้าใช้งาน
1. เปิดเว็บเบราว์เซอร์
2. ไปที่: **http://localhost:8080**
3. Login ด้วย:
   - Username: `admin`
   - Password: `admin123`

## 📋 ทดสอบระบบครั้งแรก

### 1️⃣ ทดสอบเพิ่มสินค้า
1. คลิก **สินค้า** > **เพิ่มสินค้า**
2. กรอกข้อมูล:
   - รหัส: `TEST-001`
   - ชื่อ: `สินค้าทดสอบ`
   - ราคา: `100`
   - สต็อกเริ่มต้น: `50`
   - สต็อกขั้นต่ำ: `10`
3. คลิก **บันทึก**

### 2️⃣ ทดสอบรับสินค้าเข้า
1. คลิก **จัดการสต็อก** > **รับสินค้าเข้า**
2. เลือกสินค้า `TEST-001`
3. ระบุจำนวน: `20`
4. แหล่งที่มา: `ซัพพลายเออร์ทดสอบ`
5. คลิก **บันทึกการรับเข้า**
6. ตรวจสอบว่าสต็อกเพิ่มเป็น 70

### 3️⃣ ทดสอบเบิกสินค้าออก
1. คลิก **จัดการสต็อก** > **เบิกสินค้าออก**
2. เลือกสินค้า `TEST-001`
3. ระบุจำนวน: `15`
4. ปลายทาง: `แผนกทดสอบ`
5. คลิก **บันทึกการเบิกออก**
6. ตรวจสอบว่าสต็อกลดเป็น 55

### 4️⃣ ดูประวัติ
1. คลิก **จัดการสต็อก** > **ประวัติการเคลื่อนไหว**
2. ตรวจสอบว่ามีรายการรับเข้าและเบิกออกที่บันทึกไว้

## 🎯 กรณีใช้งานจริง

### สถานการณ์ 1: รับของจากซัพพลายเออร์
```
1. ได้รับใบส่งของ PO-2025-100
2. สินค้า: เมาส์ไร้สาย จำนวน 100 ชิ้น
3. เข้าระบบ > รับสินค้าเข้า
4. เลือกสินค้า "เมาส์ไร้สาย"
5. ระบุจำนวน: 100
6. เลขที่อ้างอิง: PO-2025-100
7. แหล่งที่มา: บริษัท ABC จำกัด
8. บันทึก ✅
```

### สถานการณ์ 2: แผนกขอเบิกอุปกรณ์
```
1. แผนก IT ขอเบิกเมาส์ 10 ชิ้น
2. เข้าระบบ > เบิกสินค้าออก
3. เลือกสินค้า "เมาส์ไร้สาย"
4. ระบุจำนวน: 10
5. เลขที่อ้างอิง: REQ-IT-025
6. ปลายทาง: แผนก IT
7. หมายเหตุ: เพิ่มเครื่องใหม่ 10 เครื่อง
8. บันทึก ✅
```

### สถานการณ์ 3: ตรวจสอบสินค้าใกล้หมด
```
1. เข้า Dashboard
2. ดูที่การ์ด "สต็อกต่ำ" (สีเหลือง)
3. เลื่อนลงดู "สินค้าคงเหลือต่ำ"
4. คลิกปุ่ม ⬇️ ที่สินค้าที่ต้องการสั่งเพิ่ม
5. บันทึกการรับเข้า
```

## 🛠 การปรับแต่งระบบ

### เปลี่ยนรหัสผ่าน Admin
```sql
USE golang_crud;
UPDATE admins SET password = 'รหัสผ่านใหม่' WHERE username = 'admin';
```

### เพิ่ม Admin คนใหม่
```sql
INSERT INTO admins (username, password) VALUES ('staff01', 'password123');
```

### เพิ่มหมวดหมู่สินค้า
```sql
INSERT INTO categories (name, description) 
VALUES ('ชื่อหมวดหมู่', 'คำอธิบาย');
```
